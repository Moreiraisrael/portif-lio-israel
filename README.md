# Portfólio de Israel Santos - Editor de VSL & AI

Este é um template de portfólio moderno, otimizado para o cargo de Editor de VSL e Anúncios com foco em IA.

## Como Usar

1. **Imagens e Vídeos**:
    * Substitua as imagens em `img/` ou os links de `unsplash` no arquivo `index.html` pelas suas próprias thumbnails de projetos reais.
    * Para adicionar vídeos reais, você pode substituir as tags `<img>` dentro de `.project-thumb` por um `<iframe>` do YouTube/Vimeo ou usar a tag `<video>` do HTML5.

2. **Customização**:
    * Edite o arquivo `index.html` para atualizar os textos "Sobre Mim" e adicionar links reais para seu WhatsApp e Email.
    * No arquivo `style.css`, você pode mudar as cores principais (`--accent-color`, `--accent-secondary`) se desejar outra paleta.

3. **Tecnologias Usadas**:
    * HTML5 Semântico
    * CSS3 Moderno (Grid, Flexbox, Variáveis, Animações)
    * JavaScript (Scroll Suave, Menu Mobile, Animações ao Rolar)
